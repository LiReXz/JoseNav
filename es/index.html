Sobre mí
Soy un Ingeniero DevOps que trabaja en la construcción y gestión de infraestructura y pipelines de despliegue. He trabajado con herramientas de CI/CD como GitHub Actions y Jenkins, y uso Terraform para gestionar infraestructura como código.

Estoy familiarizado con plataformas cloud como AWS y Azure, y he trabajado con herramientas de containerización como Docker, Kubernetes, Helm y OpenShift. También tengo experiencia gestionando secretos con HashiCorp Vault y configurando flujos de trabajo usando Apache Airflow y Control-M.

Para monitorización, he usado herramientas como Prometheus, Grafana y Dynatrace para asegurar que los sistemas funcionen sin problemas. Disfruto resolviendo desafíos con scripts de Python y Bash para optimizar procesos y construir sistemas confiables.

Mi enfoque está en construir sistemas que sean escalables, confiables y mantenibles.
