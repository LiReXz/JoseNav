Project Attribution

Original Project: Ranky Discord Bot by MKGF

Live Application: ranky.top

My Contribution: Complete DevOps implementation and infrastructure automation

Repository: Available in my Collaborations repository

This collaboration involved implementing a complete AWS infrastructure solution for MKGF's Discord bot project. Ranky is a League of Legends custom ranking system that allows gaming communities to create personalized competitive experiences beyond traditional ranking systems.

My specific contribution focused on: Designing and implementing the entire AWS infrastructure with Terraform, creating CI/CD pipelines for automated deployment, containerizing the application with Docker, and optimizing everything for AWS Free Tier usage with cost-free lifecycle management.
