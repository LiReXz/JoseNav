Ranky Discord Bot

League of Legends Custom Ranking System with Automated AWS Deployment

Java Spring Boot Discord API AWS Terraform GitHub Actions Docker Maven

Project Overview

Ranky is a collaborative Discord bot project designed to create custom League of Legends rankings for communities. The bot allows players to compete in personalized SoloQ/DuoQ challenges, extending beyond traditional ranking systems to create engaging competitive experiences in smaller gaming communities.

This project showcases a complete full-stack application with automated deployment pipeline, integrating Discord API, Riot Games API, AWS cloud infrastructure, and modern DevOps practices.

Technical Architecture

Backend Application

  • Framework: Spring Boot 3.2.5
  • Language: Java 17
  • Security: Spring Security
  • Build Tool: Maven
  • API Integration: Discord & Riot Games APIs

AWS Infrastructure

  • Compute: ECS with Fargate
  • Container Registry: ECR
  • Networking: VPC, Subnets, Security Groups
  • Secrets: AWS Secrets Manager
  • Monitoring: CloudWatch Logs

DevOps Pipeline

  • CI/CD: GitHub Actions
  • Infrastructure as Code: Terraform
  • Containerization: Docker
  • Automated Testing: Maven Surefire
  • Secret Management: GitHub Secrets

Key Features

Discord Bot Integration

Full Discord bot functionality with command handling, user authentication, and interactive messaging for managing custom rankings.

Custom Ranking System

Create and manage personalized League of Legends rankings with real-time data integration from Riot Games API.

Automated Deployment

Complete CI/CD pipeline with GitHub Actions that builds, tests, and deploys to AWS ECS automatically.

Secure Configuration

API keys and sensitive data managed through AWS Secrets Manager with Spring Boot security configurations.

Deployment Pipeline

1

Secret Injection

GitHub Actions creates application-secret.properties with API keys from GitHub Secrets

2

Build & Test

Maven compiles the Java application and runs automated tests

3

Containerization

Docker builds container image and pushes to AWS ECR registry

4

Infrastructure

Terraform provisions AWS resources (ECS, VPC, Security Groups)

5

Deployment

ECS service deploys the containerized application with health checks
